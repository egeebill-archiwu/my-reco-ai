如何取得免費的 Gemini API Key？

  1. 點擊前往 Google AI Studio 。
  2. 登入您的 Google 帳號。
  3. 點擊左上角的 "Get API key" 按鈕。
  4. 建立並複製您的 API Key，然後貼在上方輸入框中保存。

RECO AI 使用說明

歡迎使用 RECO AI！這是一個專屬您的智慧會議工具。

本程式支援無限制的高精度錄音轉譯、說話者識別、重點摘要，並配備類似 NotebookLM 的 AI 筆記助理與備註卡片功能。

💡 核心功能介紹：

  1. 高精度語音轉譯：支援上傳 MP3, WAV, M4A, WebM 等。AI 自動識別說話者並產生精準時間戳。
  2. 點擊時間戳播放：點擊逐字稿中的時間戳，播放器會自動跳轉至該時間點播放，方便快速核對。
  3. AI 會議助理 (Q&A)：在右側面板中，可以詢問 AI 關於這場會議的任何細節與整理決議。
  4. 備註卡片：隨時記錄並儲存與會議關聯的個人筆記、想法與追蹤事項。

📱 手機使用小秘訣：

在手機上開會時，建議使用手機內建的「語音備忘錄」進行背景錄音（鎖屏、切換 Line 都不會中斷）。錄製完後，直接用手機打開本網頁一鍵上傳轉譯！
